# 🧠 Cuckoo Search for Heart Disease Prediction

## 📌 Project Overview

This project focuses on predicting heart disease using machine learning techniques combined with an optimization algorithm called **Cuckoo Search**. The goal is to select the most relevant features from the dataset to improve model performance.

---

## 🎯 Objective

* To perform **feature selection** using Cuckoo Search Optimization
* To improve prediction accuracy of heart disease
* To compare performance using machine learning models

---

## 🧪 Technologies Used

* Python
* NumPy, Pandas
* Scikit-learn
* Optimization Algorithm: Cuckoo Search

---

## 📊 Dataset

* UCI Heart Disease Dataset
* Includes multiple processed datasets such as:

  * Cleveland
  * Hungarian
  * Switzerland
  * VA dataset

---

## ⚙️ Methodology

1. Data preprocessing and cleaning
2. Feature selection using Cuckoo Search algorithm
3. Model training using Random Forest
4. Performance evaluation

---

## 📁 Project Structure

* `dataset/` → Contains all dataset files
* `preprocessing.py` → Data cleaning and preparation
* `feature_selection.py` → Feature selection logic
* `cuckoo_search.py` → Optimization algorithm
* `randomforest.py` → ML model
* `main.py` → Main execution file

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
```

---

## 📈 Results

* Improved model performance using optimized feature selection
* Reduced redundant features

---


## 📌 Conclusion

Cuckoo Search Optimization effectively enhances feature selection, leading to better performance in heart disease prediction models.
